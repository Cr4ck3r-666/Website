File menager

Strawberry file menager is It's simple terminal file meneger for Linux base systems writen in C++ and some Python. Meneger is default in red color theme but You can change it. My main priority is basic user friendly terminal intarface. Someone people has problem with learning Linux terminal and my proposition for it is give sipmlifyied commands of Linux command's like del (rm -rf on Linux) or list (ls on Linux) + some dedicated commands like fsize ,img or mkd. In creation process of command name's I was inspired by Basic programing languige who has similar commands.

8-BIT COMPUTER

(Work in progress)

In all time when I was making electronic stuff (9 maybe 10 years) I have a once dream. That's dream is my own diy computer and that's a reason why I start doing electronic. And now is the moment for realasing this dream.

1. Idea

This computer idea is very simple. Make a simple computer with 8-Bit procesor ,max 1MB ram and slot's for perypherial device - like IBM PC class. Optionaly be added a batery power sublying (9-12V) ,becouse I have in plans make "SDR card" for this computer and give it somewhere. I have many ic's for that's and other intresting perypherial cards.

2. Specyfication

Procesor: NEC D8086

UV-EPROM: ST M27C1001

S-RAM: AMD AM39L422DMB

3. SRAM Memory and why is on another board?

Ram memory is on another noard ,becouse I don't have many pleace on motherboard and at moment when I build a motherboard I don't have this ram.

ARDUINO PWM GENERATOR

Today i was made a very simple PWM generator based on arduno micro to my lab. It was be powered from usb connector and sygnalised a modulation using 10 led's. One led is 10% of modulation ,2 led's is 20% ,3 led's is 30% ,ect. Previously instead led's I wanted to use once of my 7 segments display and second arduino as i2c driver for it but ultimately build it on leds. Case of this generator is made with case from old VHS cassete of Polish cartoon "Pomysłowy dobromir".
